IB Pseudocode Tool by Mats Macke

This website is a tool to develop and test IB Pseudocode in order to practice for IB Computer Science exams. This has been possible in the past with Mr Dave Mulkey's EZ Pseudocode, but most students agree that tool is not very pleasant to use. This tool should be nicer to use. I am aware that my compiler is by no means perfect yet, and some programs may cause bugs. Please do report these by adding an issue on the GitHub repository, linked below.
